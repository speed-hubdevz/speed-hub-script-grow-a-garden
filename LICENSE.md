## 🛠️ Troubleshooting Common Issues

- ❌ **Script does not execute after pasting**
  The executor may be outdated or missing key API calls. Update your executor to the latest version and retry. Confirm you are pasting the full script payload without truncation.

- ❌ **GUI does not appear in-game**
  This usually means the injection happened before the game fully loaded. Wait until you are standing in your garden, then re-execute the script. On Android, ensure overlay permissions are granted.

- ❌ **Auto-Plant has no visible effect**
  Check that `AutoPlantEnabled` is set to `true` in your config. If using the GUI slider, make sure you have seeds in your inventory.

- ❌ **Auto-Harvest only collects some crops**
  The **Auto-Harvest** module fires on a single tick. If new crops mature immediately after, they will not be caught in that cycle. Rebind the key and tap it again after each growth cycle.

- ❌ **Speed Modifier has no effect**
  Ensure `SpeedMultiplier` is set above `1.0` in your config. Some private servers disable certain physics calls which can block the speed module.

- ❌ **Auto-Rejoin sends you to a different server**
  Roblox occasionally rotates server IDs during maintenance windows. Disable auto-rejoin during known update periods and re-enable it once servers stabilize.
